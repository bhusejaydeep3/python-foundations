# OOP Projects — Python

A collection of Object-Oriented Programming projects built in Python
covering all four OOP pillars: Abstraction, Encapsulation, 
Inheritance, and Polymorphism.

## Projects

### 1. Banking System

## Concepts Covered
- Abstract Base Classes (ABC)
- Encapsulation with private variables
- Inheritance via super()
- Polymorphism via method overriding

- Python 3
- NumPy
- abc module
